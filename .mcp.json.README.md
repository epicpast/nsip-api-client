# MCP Configuration

## Setup

1. Copy the example file:
   ```bash
   cp .mcp.json.example .mcp.json
   ```

2. Edit `.mcp.json` and add your API keys:
   - **context7**: Get an API key from https://upstash.com/
   - **Other services**: Configure as needed

3. **NEVER commit `.mcp.json` to version control!**
   - It's already in `.gitignore`
   - It contains sensitive API keys and personal paths

## Security Note

If you accidentally commit `.mcp.json`:
1. Remove it immediately: `git rm --cached .mcp.json`
2. Add to .gitignore (already done)
3. Rotate your API keys immediately
4. Consider rewriting git history if pushed to remote

## MCP Servers Configured

- **context7**: Library documentation retrieval
- **chrome-devtools**: Browser automation and debugging
- **aws-documentation**: AWS documentation access
- **codex**: Deep reasoning and research tasks
- **obsidian-mcp-tools**: Personal Obsidian vault integration (optional)
